# Sales Performance Analyzer (Python Project)

A Python project that analyzes sales data using **Python** with Pandas usage.  
Calculates revenue, product-wise and region-wise performance, and identifies top-selling products and regions.

---

## Objective
- Calculate total revenue  
- Analyze revenue by product  
- Analyze revenue by region  
- Identify top-selling product and top-performing region  

---

## Dataset
- File: `sales_data.csv`  
- Columns: Date, Product, Region, Quantity, Price   

---

## Features
- Total revenue calculation  
- Product-wise revenue aggregation  
- Region-wise revenue aggregation  
- Top-performing product and region detection  
- Exception handling for missing files  

---

## How to run
1. Place `sales_data.csv` in the same folder  
2. Run the script: `python sales_performance_analyzer.py`  

---
